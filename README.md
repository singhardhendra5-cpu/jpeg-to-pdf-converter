# JPEG to PDF Converter

A simple and efficient tool for converting JPEG images to PDF format.

## Features

- Convert single or multiple JPEG images to PDF
- Maintain image quality during conversion
- Support for various image formats
- Batch processing capabilities
- Easy-to-use interface
- Lightweight and fast conversion

## Installation

```bash
git clone https://github.com/singhardhendra5-cpu/jpeg-to-pdf-converter.git
cd jpeg-to-pdf-converter
npm install
```

## Usage

```bash
npm start
```

Follow the on-screen instructions to select JPEG images and convert them to PDF.

## Requirements

- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher)
- Pillow or similar image processing library

## Features & Benefits

- **Fast Conversion:** Quick and efficient conversion process
- **Quality Preservation:** Maintains image quality throughout the conversion
- **Batch Processing:** Convert multiple images at once
- **User-Friendly:** Simple and intuitive interface
